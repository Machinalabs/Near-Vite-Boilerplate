# Chain Abstracted Memecoin Launchpad

This is a launchpad for memecoin that is chain agnostic and allow users to deploy to any EVM chain.

This initial version only supports:

- Base testnet

- Arbitrum testnet

## Getting Started

To run the example:

```bash
$ git clone <REPO>

$ nvm use

$ pnpm install

$ pnpm run dev
```
